# L'application Home

Elle sert à l'interface commerciale (site web de l'entreprise) mais aussi 
de point d'entré au comptes des utilisateur de l'application Bappiment.  
Les routes sont donc liées à la gestion des comptes utilisateurs ainsi que de 
leur statut dans l'entreprise. chaque statut possède donc ses informations 
liées et n'est donc pas parasité par les autres: le magasinnier ne verra pas 
les mêmes infos que le comtable, en revanche le comptable peut jeter un oeil 
sur le stock afin d'en comptabiliser les dépenses.
