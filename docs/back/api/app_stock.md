# L'application Stock

Elle sera en majeur partie utilisé par le magasinier mais pourra aussi 
faire l'objet de traitements liés à la signature d'un devis ou à l'édition 
d'un bon de travail lorsque par exemple un dépannaur aurra remplacé un organe 
sur une installation.    
Chaque camion sera aussi tracké par l'application afin de savoir quels produits 
sont présents dans quels véhicule afin d'optimiser les cas de 
rapprovisionnement.
